Some things to do to harden a new Firefox installation.

## Extensions
- uBlock Origin
- ClearURLs
- LocalCDN
- Cookie AutoDelete
- Facebook Container

## Settings
- HTTPS-only mode in all windows
- Disable search suggestions
- Set default search engine to DuckDuckGo
